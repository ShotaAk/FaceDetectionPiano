# FaceDetectionPiano

This program detects human faces from camera capture images, and plays the piano.


## Software Requirements
* python (2.7.6)
* OpenCV (2.4.8)
* dlib (19.4.0)
* pygame  (1.9.1)
* skimage (0.13.0)

## Usage
    1. Connect a camera to PC
    
    $ python piano.py
    push 'q' to exit
